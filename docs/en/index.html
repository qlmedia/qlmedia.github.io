QLMedia

QuickLook macOS extension for video/audio preview.

General Information

  • Minimal macOS version: 10.15 (Catalina), earlier versions lack required functionality.
  • Activating the extension requires running QLMedia.app at least once.
  • App Store link: https://apps.apple.com/app/qlmedia/id1667620638
